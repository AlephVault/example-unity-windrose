Detailed Description

A prompt builder lets you build an array of Prompt elements to be passed to Interactors.Interactor.RunInteraction(InteractiveMessage, Prompt[]).

It starts containing an empty array and then you can call the methods of this class to add elements to the array following a fluent or chained pattern. To obtain the final array you simply call End method when done adding stuff.
