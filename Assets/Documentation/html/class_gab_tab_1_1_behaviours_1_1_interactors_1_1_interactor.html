Detailed Description

An interactor is an UI component that has a special method: RunInteraction(InteractiveMessage, InteractiveMessage.Prompt[]). Yow will make use of standard subclasses, or create your own.

Interactors make sense only in the scope of an InteractiveInterface. Interactors are registered (in the editor) inside an InteractorsManager instance, and accessed/used as in the example given in InteractiveInterface.RunInteraction(Func<InteractorsManager, InteractiveMessage, IEnumerator>).

The only relevant method of this class is RunInteraction(InteractiveMessage, InteractiveMessage.Prompt[]), which invokes Input, that must be implemented.
