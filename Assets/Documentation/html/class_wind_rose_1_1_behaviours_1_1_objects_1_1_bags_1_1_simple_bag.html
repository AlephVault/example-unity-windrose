Detailed Description

Simple bags are intended to be used as portable bags on objects and they will handle just ONE container, but with convenience methods to abstract the user regarding the inners of the single-container positioning strategy. Aside from that, methods to drop and pick an object to/from the DropLayer.

They are tightly related to InventoryManagementStrategyHolder and InventorySimpleBagRenderingManagementStrategy.
