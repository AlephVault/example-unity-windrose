Detailed Description

A stack strategy will be barely more than a simple data bundle: in many cases it will not hold logic on its own except for non-side-effected logic that would compute data without altering anything.

Stack strategies are created by corresponding item strategies, and so they will return appropriate reference to their creators.It will also hold a reference to the stack it is bound to.
