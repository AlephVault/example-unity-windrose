Detailed Description

This class provides an interface to interact via UI with the game, with stuff like messaging and custom components.

The interface is quite like those in Pokemon: one text where the game or NPCs tell you everything, and also UI elements to interact with (e.g. buttons set, text input, or list of elements). It require several components:

This behaviour exposes a method to run this magic: RunInteraction(Interaction func). The interaction must be a generator function(remember that C# does not allow anonymous functions working as generators). For more information, see RunInteraction method's documentation in this class.

This component requires an UnityEngine.UI.Image, which will hold the background. The recommended settings for the image is:

  • Image Type Slice
    • Fill Center Set it to true

This component requires an Hideable, which will hide the interface when not being used.

This component requires an Interactors.InteractorsManager, which will manage the available components of this interactive interface.

Assigned via editor, an object with InteractiveMessage must be present. Usually, this object will exist among the direct children of this component, but it is not mandatory.

◆ RunInteraction()

Coroutine GabTab.Behaviours.InteractiveInterface.RunInteraction ( Func< Interactors.InteractorsManager, InteractiveMessage, IEnumerator >  runnable)

This method runs an interaction procedure.

An interaction procedure is a function that returns an IEnumerator (usually, this is a generator function - those functions with yield return or yield break statements). Remember: Anonymous functions cannot be used as generators.

Such function will yield steps of flow. See the example to get a glance of how does this work.

Events beforeRunningInteraction and afterRunningInteraction will be trigger respectively to wrap each passed interaction.

Parameters
runnableThe function to pass arguments. The function will accept an Interactors.InteractorsManager as first argument and an as second argument.

class MyClass { void Awake() { interactiveInterface = GetComponent<InteractiveInterface>(); }

IEnumerator SampleInteraction(Interactors.InteractorsManager manager, InteractiveMessage message) { ButtonsInteractor yesnoInteractor = (ButtonsInteractor)manager["yesno-input"]; NullInteractor nullInteractor = (NullInteractor)manager["null"]; yield return yesnoInteractor.RunInteraction(message, new InteractiveMessage.PromptBuilder().Clear().Write("Do you like GabTab?").End()); if (yesnoInteractor.Result == "yes") { yield return nullInteractor.RunInteraction(message, new InteractiveMessage.PromptBuilder().Clear().Write("Cool!").End()); } else { yield return nullInteractor.RunInteraction(message, new InteractiveMessage.PromptBuilder().Clear().Write("Ooooh :(").End()); } }

void RunSampleInteraction() { interactiveInterface.RunInteraction(SampleInteraction); } }
