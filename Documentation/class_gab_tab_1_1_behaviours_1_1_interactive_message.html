Detailed Description

This component handles the message being displayed to the user. It is intended to be a child of an component.

It requires two components in the same object, and one component in a child. The components go like this:

  1. This component has also recommended settings since it inherits from ScrollRect: > Viewport: None > Horizontal Scrollbar: None > Vertical Scrollbar: None
  2. InteractiveMessageContent (in child). It is already documented.

This class provides features to display text in an animated fashion instead of suddenly the entire bulk. An example is provided in InteractiveInterface.

Pay special attention to PromptMessages(Prompt[]) since it is the most interesting method of this class, aside from the inner classes PromptBuilder and Prompt.

This component should:

  • Configure its Image dependency appropriately:
    • Image Type Slice
      • Fill Center Set it to true
  • Configure its Mask dependency appropriately. Since the mask is used to clip the text content while it scrolls, it will be needed here as well and appropriately configure:
    • Show Mask Graphic Set it to true or the background image for the text content will not appear.
  • Configure this component, unless you want a different behaviour, like this:
    • Viewport None.
    • Vertical scrollbar None.
    • Horizontal scrollbar None.
    • Message Content A valid messageContent instance. Usually among its direct children.
