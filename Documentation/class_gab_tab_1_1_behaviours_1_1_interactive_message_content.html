Detailed Description

This component reflects the settings of text speed and explicit wait speed.

It also provides methods used by InteractiveMessage to yield timing coroutines to perform the dirty job for us. Such coroutines are started in terms of this object's configuration.

You should configure this object's components as well. Recommended settings are:

  • Text This component will ultimately display the text. The recommended settings are:
    • Paragraph
      • Alignment Left and Top
      • Horizontal Overflow Wrap
      • Vertical Overflow Overflow
    • Character
      • Line Spacing 1
  • Content Size Fitter This component is used to scroll appropriately. Recommended:
    • Horizontal Fit Unconstrained
    • Vertical Fit Preferred Size
