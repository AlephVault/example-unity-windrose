Detailed Description

Simple drop containers are components that render the loot in a single square. Loot is a sorted stack of item stacks.

This component is not alone: The same game object must have at least one instance of SpriteRenderer and it MUST be turned into a prefab to be used. The use of a SortingGroup will help us sorting the sprites appropriately.

The sprites will render the contents in stacking order: the "last" item in the container will be rendered topmost, while the "first" one will be rendered last, or hidden in the "bulk": If there are more items to render, than sprites, the last sprite will not render a regular item but instead a sprite told to be a "generic" image (say, the picture of a dropped purse).

This renderer is only useful for simple rendering: stacks consisting on sprite, quantity and caption, located inside drop containers at integer positions.
