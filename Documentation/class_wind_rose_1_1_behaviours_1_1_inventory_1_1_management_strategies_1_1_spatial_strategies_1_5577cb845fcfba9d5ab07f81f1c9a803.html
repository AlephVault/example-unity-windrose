Detailed Description

Spatial management strategies manage the position of the elements in the containers.

There may be different implementations like standard indexed inventory (e.g. Baldur's Gate) or matrix inventory with items of different sizes (e.g. Neverwinter Nights).

Spatial management strategies will involve handling a particular type of position values (e.g. integer index, or vector position), and a particular type of inner container, to run the appropriate logic to find or add items therein.
