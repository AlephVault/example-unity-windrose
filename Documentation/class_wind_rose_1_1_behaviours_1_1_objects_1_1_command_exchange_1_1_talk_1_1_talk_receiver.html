Detailed Description

This class is usually attached to NPCs. It will receive the TalkSender.COMMAND sent by an TalkSender and trigger the onTalkReceived event.

When receiving the command, this object will look towards the opposite direction the sender object is looking to, and then trigger the onTalkReceived event.
