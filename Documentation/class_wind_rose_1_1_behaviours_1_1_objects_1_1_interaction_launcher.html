Detailed Description

This component's sole purpose is to retrieve the topmost UI.InteractionProvider instance. This, to be able to trigger (launch) any interaction using such retrieved component.

To understand the role of UI.InteractionProvider and why such component must be in the topmost object of the current WindRose level objects' hierarchy, see that class' documentation and then come back.

The spirit of this component is that it may provide a mean for other components (perhaps reacting to events in TriggerZone or related classes) to retrieve the interaction provider and start an interaction with the player.
