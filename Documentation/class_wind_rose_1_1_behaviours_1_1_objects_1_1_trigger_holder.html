Detailed Description

This behaviour configures a collision mask (giving it appropriate dimensions and and making it a trigger).

Although the size and pivot are determined once, and this behaviour does not touch the collider or handles the collisions, it will provide a method named RefreshDimensions so other users can make use of it when needed.
