Detailed Description

This components sets its underlying collider (which will be a BoxCollider2D) to the dimensions of the object (considering underlying cell's width/height).

Live triggers are intended to be installed into moving objects, and not in objects that intend to be like platforms. They ensure the collider is only active while the object is added to a map.
