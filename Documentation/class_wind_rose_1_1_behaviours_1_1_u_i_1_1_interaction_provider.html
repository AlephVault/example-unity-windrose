Detailed Description

Wraps all the map in the scene and also the InteractiveInterface that is available inside. It is expected that only one be available.

The wrapped interactive interface will also have two listener to, perhaps, pause/release the maps. This interface will be available for any child that would need it (e.g. to start an interaction).
