Detailed Description

A map layer sorts itself inside its parent map, in the DEFAULT layer, and in the zero point, identity rotation, and identity scale. Map layers will hold different aspects of the map's content, and while currently only 4 layer types are supported, nothing prevents the development of more layer types (although this is entirely up to the developer).
