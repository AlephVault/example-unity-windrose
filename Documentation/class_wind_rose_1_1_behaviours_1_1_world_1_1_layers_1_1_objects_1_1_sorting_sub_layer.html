Detailed Description

Sorting sub-layers depend on the map and create several children holding a SortingGroup component each one. Since the number of sorting groups being created depends on the map this object belongs to, this implies this object is required to be inside a map (it will be usually bound to a layer), but it will actually be inside a map's ObjectsLayer (actually, this component will be attached to a newly instantiated object by an ObjectsLayer).

Related to those groups, this class provides methods to get any of these groups (e.g. to be used as a parent of another object). Such group objects will be ready right on Awake.
