Detailed Description

This class defines a WindRose animation spec. An animation spec contains the frames to render and the amount of intended frames per second. Usually, this class is used inside an animation set (for certain direction in such set) or as default animation for objects that are not orientable (i.e. they have no direction to look/move to).
