Detailed Description

Bitmasks (or bitmaps) are bidimensional array of bits. They are usually intended for per-map logic involving one boolean value on each (x, y) coordinate.

Member Enumeration Documentation

◆ CheckType

enum WindRose.Types.Bitmask.CheckType
strong

Check types are used when checking several cells together. They return the kind of aggregate value to return. To this extent, we consider that "blocked" is true and "free" is false, for historical reasons.

Consider the four aristotelic quantifiers for predicates. They are, actually, those operations.

Bitmask WindRose.Types.Bitmask.Translated ( uint  newWidth,
uint  newHeight,
int  offsetX,
int  offsetY,
bool  newFillingValue = false 
)

Creates a new bitmask based on this one, but modified in size and/or translated in offset.

Think as if you do it manually using old versions of MS Paint for a monochrome picture.

First, you will create your new picture, filling with full black or full white.

Then you go to your former picture and select-all and copy.

Then you go back to this picture, select the rectangle-selector, and place somewhere a tiny 1x1 selection.

Then you paste the content, choosing to not modify the dimensions of the drawing if it doesn't fit but instead clip the pasted content.

Parameters
newWidthThe width of the new bitmask.
newHeightThe height of the new bitmask.
offsetXThe X offset on which the current content will be pasted.
offsetYThe Y offset on which the current content will be pasted.
newFillingValueThe initial value for the destination bitmask.
Returns
The new bitmask.
