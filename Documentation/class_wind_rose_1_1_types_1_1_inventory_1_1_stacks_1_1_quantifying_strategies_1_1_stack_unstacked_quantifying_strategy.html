out object  quantityAdded,
out object  quantityLeft 
)
virtual

Computes overflow considering whether both quantities are true.

User has to expect that each argument here is a boolean value, overflow is finding two true values, and quantity being added / left are boolean values as well: they serve well as stacks of 1 element.

Parameters
quantityThe quantity to add
finalQuantityThe quantity that would remain on the strategy
quantityAddedThe quantity that would be effectively added
quantityLeftThe quantity that would be not added
Returns
Whether it would overflow or not
